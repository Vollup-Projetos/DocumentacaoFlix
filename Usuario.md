## Procurar Usuário

**Perfis Autorizados:** Seguradoras, Parceiros

**Endpoint**: `/api/usuarios/{id}

**Método**:  `GET`

**Parâmetros:** 

- *id* - ID do usuário

**Resultado**: 

```json
{
    "USU_NOME":"Nome",
    "USU_SOBRENOME":"Sobrenome"
}

```
